# 横浜国立大学管弦楽団 公式ウェブサイト

横浜国立大学管弦楽団の公式ウェブサイトです。

## 機能

- 団紹介
- 演奏会情報
- 練習案内
- お問い合わせ
- サイト内検索

## 技術スタック

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Vercel

## 開発環境のセットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

[http://localhost:3000](http://localhost:3000) でアプリケーションを確認できます。

## デプロイ

Vercelで自動デプロイされています。
